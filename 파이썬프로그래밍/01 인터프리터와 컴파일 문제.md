# 1주차 인터프리터 언어 VS 컴파일 언어 문제

### 1. 실행 방식

인터프리터 언어는 (      ) 해석하며 실행하지만, 컴파일 언어는 전체 소스 코드를 컴파일 후 실행합니다.

- 한 줄씩
- 실행 후
- 전체 코드
- 일부만

### 2. 에러 발견 시점

인터프리터 언어는 (      )에 에러를 발견하지만, 컴파일 언어는 컴파일 시점에 에러를 발견합니다.

- 코드 작성 시
- 실행 중
- 컴파일 전
- 코드 최종 제출 시

### 3. 대표 언어

컴파일 언어의 대표적인 예는 (            ) 입니다.

- Python, JavaScript
- HTML, CSS
- C, C++
- SQL, NoSQL
