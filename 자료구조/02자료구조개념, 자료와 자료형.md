# 2주차 자료구조의 개념, 자료와 자료형

### 자료구조(data structure)

🔸 프로그램에서 자료를 효율적으로 사용하기 위한 논리적인 구조

🔸 자료의 특성에 따라 분류하고 구성하여 프로그램에서 활용할 수 있는 저장 및 처리의 체계를 제공

### 자료구조(data structure)의 기본 개념

- 자료구조를 구현한 대표적인 예 : data type
- **program = data structure + algorithm**

### 자료와 자료형

🔸 기본 데이터 타입 : built-in type(primary data type)

char : 1바이트로 하나의 문자를 표현

int : short, long unsigned

float : single precision

double : double precision

🔸 변수 선언

기억 공간 확보, 변수이름(identifier)지정, 초기화

🔸 user-defined type

자료구조를 생성하여 적용

### 교재 1장 연습문제 10번(54p)

```c
#include <stdio.h>

int main()
{
	printf("int size = %u\n", sizeof(int));
	printf("float size = %u\n", sizeof(float));
	printf("double size = %u\n", sizeof(double));
	printf("char size = %u\n", sizeof(char));

	return 0;
}
```

### 교재 1장 연습문제 11번(55p)

```c
#include <stdio.h>

int main()
{
	printf("%c의 ASCII 값 = %u\n", 'A', 'A');
	printf("%c의 ASCII 값 = %u\n", 'a', 'a');
	printf("%c의 ASCII 값 = %u\n", '0', '0');
	printf("%c의 ASCII 값 = %u\n", ' ', ' ');
	printf("%c의 ASCII 값 = %u\n", '*', '*');
		return 0;
}
```

```c
// #define _CRT_SECURE_NO_WARNINGS -> Visual Studio 사용 시 필요

	int n1, n2, n3, large;

	printf("Enter three integers : ");
	scanf("%d %d %d", &n1, &n2, &n3);

	if (n1 > n2) large = n1;
	else large = n2;

	if (n3 > large) large = n3;

	printf("The largest value : %d\n", large);

	return 0;
}
```
